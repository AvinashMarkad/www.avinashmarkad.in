The site is built from the ground up using core web technologies to ensure maximum performance, zero bloat, and a premium user experience.

---

## ✨ Features & UI/UX Highlights

I focused on creating a "High-End Aesthetic" using custom CSS and vanilla JavaScript, intentionally avoiding heavy frontend frameworks for this static site.

* **Glassmorphism Design:** Custom frosted-glass effects (`backdrop-filter`) applied to navigation and project cards for a modern, sleek interface.
* **Performant Animations:** Features pure CSS background blob animations and smooth `@keyframes` floating effects that don't compromise page load speeds.
* **Scroll-Triggered Reveals:** Implemented a modern `IntersectionObserver` in vanilla JavaScript to create high-quality, staggered reveal animations as the user scrolls.
* **Dynamic Typing Effect:** A custom JavaScript typing animation in the hero section cycling through my core specializations.
* **Fully Responsive:** Custom CSS media queries ensure a flawless experience across desktop, tablet, and mobile devices, including a custom mobile hamburger menu.
* **SEO Optimized:** Includes Schema.org markup, canonical URLs, and Open Graph tags for optimal search engine visibility.
